##### [**Dark Universe - Takeover**](https://www.nexusmods.com/starfield/mods/11045) **(Nexus Mods)**

By: kaosnyrb



Adds 50 new procedurally generated Spacer Military Bases, supporting bounties.

###### 

###### How does it work?

The bases are made up of parts which are combined by a program I've written called Startiller. These are then output to a format that Starfield understands. The POI's are exactly like the vanilla ones, there is no scripting or dynamic changes that happen. They use the same enemy references as vanilla so any mods affecting spacers will also work in the new POI's. This makes them very compatible with other mods.

###### 

###### How to find them?

Bounty board missions will start offering missions to these POI's on the next refresh (Might need to rest for a while to get new ones).They've been tweaked so as not to be the only missions available but to show up reasonable often.

They also are visible from space during scanning and will start appearing in unexplored systems once you install this mod.

Finally you can just land anywhere and you've a chance of coming across one.
